# SohDB
簡単なDBを気が向いたときに作っています。<br>
C言語を長らく使っていないので忘れないようにやってます。

## 環境構築
ubuntu にて実行しています。
1. ubuntuをインストール
2. `sudo apt-get update`
3. `sudo apt-get install`
4. `mkdir programdata`
5. `cd programdata`
6. `gcc --version`
7. gccが入っていなかった場合、指示にしたがってgccをインストールしてください
8. `git clone ???`
9. `gcc -o sohdb.exe main.c`
実行時は`./sohdb.exe`で起動してください
